== Description ==

This plugin generate an acrostic in french based on a first name.
Ce plugin fait des acrostiches en français. A partir d'un prénom il donne une liste de qualités qui commence par chaque lettre du prénom.

== Installation ==

1. Upload the `acrostiche` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the shortcode `[acrosticheForm]` to display the acrostiche generator on a page or post.
